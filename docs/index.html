Frequently Asked Questions

How do I bulk import subitems into Monday.com? ▾

Install the Monday.com Inspector Chrome extension, navigate to your board, upload your CSV or Excel file, map the columns, and click Import. The tool creates subitems under the correct parent items automatically.

Can I import parent items and subitems at the same time? ▾

Yes. Monday.com Inspector supports two-phase import: it first creates parent items with their column values, then creates subitems underneath them. You can also import subitems only if the parent items already exist.

Does this work with Monday.com board exports? ▾

Yes. The extension auto-detects Monday.com's hierarchical XLSX export format, preserving groups, parent items, and subitems. You can export from one board and re-import into another.

Is the Monday.com Inspector extension free? ▾

Yes, the extension is completely free and open source. There are no row limits, no account required, and your API token stays securely in your browser.

What column types are supported? ▾

Monday.com Inspector supports all writable column types including text, numbers, status, people, date, timeline, dropdown, checkbox, link, and more. Read-only columns like formulas and mirrors are automatically excluded.

How do I import CSV data into Monday.com subitems? ▾

Prepare your CSV with columns matching your Monday.com subitem columns. Install the extension, enter your API token and board ID, upload the file, map CSV columns to board columns, and click Start Import. The tool handles batching and rate limits automatically.

Can I import Excel files into Monday.com? ▾

Yes. The extension supports both .xlsx and .csv files. It also recognizes Monday.com's native board export format, so you can re-import exported boards directly.
