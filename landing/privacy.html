Monday.com Inspector Add to Chrome

Privacy Policy

Last updated: March 14, 2026

Overview

Monday.com Inspector ("the Extension") is a Chrome extension that helps users bulk import parent items and subitems into Monday.com boards from CSV or Excel files. Your privacy is important to us. This policy explains what data the Extension accesses, how it is used, and how it is stored.

Data We Collect

We do not collect, transmit, or store any personal data on external servers.

The Extension operates entirely within your browser. Specifically:

  • Monday.com API Token: Stored locally in your browser using chrome.storage.local. It is never sent anywhere except directly to the Monday.com API (https://api.monday.com/v2) to perform the import operations you initiate.
  • Uploaded Files (CSV/Excel): Parsed entirely in your browser's memory. File contents are never uploaded to any server.
  • Usage Counters: A simple numeric counter of completed imports is stored locally in your browser to display helpful prompts (e.g., a review reminder). This data never leaves your device.

Data We Do NOT Collect

  • No personal information (name, email, address)
  • No analytics or tracking data
  • No cookies for advertising or profiling
  • No browsing history
  • No data shared with third parties

Third-Party Services

The Extension communicates only with the Monday.com API (https://api.monday.com/v2) using the API token you provide. No other third-party services, analytics platforms, or advertising networks are used.

Permissions

The Extension requests the following Chrome permissions:

  • storage: To save your API token and usage preferences locally in your browser.
  • host_permissions (https://*.monday.com/*): To inject the "Import Subitems" button into the Monday.com interface and communicate with the Monday.com API.

Data Retention

All data is stored locally in your browser via chrome.storage.local. You can clear this data at any time by:

  • Removing the extension from Chrome
  • Clearing your browser's extension storage

Changes to This Policy

If we update this privacy policy, the revised version will be posted on this page with an updated "Last updated" date. Continued use of the Extension after changes constitutes acceptance of the updated policy.
